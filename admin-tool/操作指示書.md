## 1. ツール概要

**やること**: Notion で作成した記事をコーポレートサイト（a-inc.info）に公開する

**自動で行われること**:
- Notion 本文のHTML変換・ページ生成
- 記事一覧（howto/posts、howto BOX）への自動追加
- 更新日の自動記録（Notion未指定の場合は当日日付）
- Cloudflare Pages へのデプロイ
- Notion ステータスの自動更新（→「掲載完了」）

---

## 2. 事前準備（初回のみ）

### ターミナルを使えるようにする

macOS の「ターミナル」アプリを開いてください（アプリケーション → ユーティリティ → ターミナル）。

### 必要なソフトウェア確認

```bash
node -v   # v18以上が表示されればOK
```

表示されない場合は担当エンジニアに連絡してください。

## 3. 管理ツールの起動方法

### ① ターミナルを開いてディレクトリに移動

```bash
cd /Users/asamin/CascadeProjects/corporate-website/admin-tool
```

### ② サーバーを起動

```bash
npm start
```

以下のメッセージが表示されれば起動成功：

```
記事アップロード管理ツール起動中: http://localhost:3001
✅ [起動チェック] insightsData: ローカル = デプロイ済み (XX件)
✅ [起動チェック] articlesData: ローカル = デプロイ済み (XX件)
```

> ⚠️ **起動チェックで「差異があります」と表示された場合**  
> ローカルとサイトの記事データにズレがある可能性があります。担当エンジニアに確認してください。

### ③ ブラウザでアクセス

`http://localhost:3001` を開く

### ④ 終了する場合

ターミナルで `Ctrl + C` を押す

## 4. Notionでの記事準備

### DB の場所

Notion 内の「記事コンテンツ管理DB」を開いてください。

### 記事に設定する項目

| 項目 | 説明 | 例 |
|------|------|----|
| **記事タイトル** | ページタイトルとして表示 | ECで成果が出る企業が... |
| **記事ID（ハンドル名）** | URLの一部になる英数字 | `b-1`、`article5` |
| **大カテゴリ** | `howto` / `insights` / `manual` から選択 | `insights` |
| **小カテゴリ** | 大カテゴリに応じて選択（下表参照） | `web-marketing` |
| **ステータス** | `デプロイ待ち` に設定するとツールに表示される | `デプロイ待ち` |
| **更新日** | 空欄の場合はデプロイ当日の日付が自動入力される | 空欄でOK |

### 小カテゴリの選択肢

| 大カテゴリ | 小カテゴリ |
|------------|------------|
| `howto` | `article` |
| `manual` | `wix` / `shopify` |
| `insights` | `ec-growth` / `web-marketing` |

### Notion 本文のフォーマット

本文は以下のセクション構造で記述してください（GPTs が自動生成する場合も同様）：

```
## META
（記事の要約情報）

## TOC
（目次）

## BODY
（本文）

## IMAGES
（画像指示）
```

#### META セクションの書き方

```
excerpt: 記事の概要文（一覧ページに表示される2〜3行の説明）
tags: ["タグ1", "タグ2", "タグ3"]
date: 2026.06.14   ← 省略可（省略するとデプロイ当日が自動入力）
```

#### IMAGES セクションの書き方

```
thumb: b-1_thumb.webp   ← サムネイル（必須）
section1: b-1_section1.webp
section2: b-1_section2.webp
```

## 5. 画像の準備と格納

### 画像の仕様

| 項目 | 推奨設定 |
|------|----------|
| サムネイル形式 | WebP または PNG |
| サムネイルサイズ | 横 1200px 以上、比率 16:9 推奨 |
| セクション画像 | 横 1200px 以上 |

### 画像の格納先

大カテゴリ・小カテゴリによってフォルダが異なります：

| 大カテゴリ | 小カテゴリ | 格納先フォルダ |
|------------|------------|----------------|
| `insights` | `web-marketing` | `/assets/images/insights/web-marketing/` |
| `insights` | `ec-growth` | `/assets/images/insights/ec-growth/` |
| `howto` | `article` | `/assets/images/howto/article/` |
| `manual` | `wix` | `/assets/images/manual/wix/` |
| `manual` | `shopify` | `/assets/images/manual/shopify/` |

### ファイル名の命名規則

- サムネイル: `{記事ID}_thumb.webp`（例: `b-1_thumb.webp`）
- セクション画像: `{記事ID}_section1.webp`、`{記事ID}_section2.webp` …

### 格納方法

Finder でフォルダを開き、画像ファイルをドラッグ＆ドロップして配置してください。

## 6. デプロイ手順（記事を公開する）

### STEP 1: ツールを起動してブラウザを開く

[第3章](#3-管理ツールの起動方法) の手順でツールを起動し、`http://localhost:3001` を開く。

---

### STEP 2: 記事一覧から対象記事を選択

ツール画面に「デプロイ待ち」ステータスの記事が一覧表示されます。  
対象の記事の **「詳細・デプロイ」** ボタンをクリックしてください。

---

### STEP 3: バリデーション結果を確認

記事の詳細画面でバリデーション結果が表示されます：

| 表示 | 意味 | 対処 |
|------|------|------|
| ✅ エラーなし | 問題なし | そのまま進める |
| ⚠️ 警告 | 軽微な不備（画像不足など） | 確認してから進める |
| ❌ エラー | 重大な不備 | Notion 本文を修正して再読み込み |

---

### STEP 4: プレビュー確認

**「プレビュー生成」** ボタンをクリックし、記事のプレビューを確認してください。

チェックポイント：
- [ ] タイトルが正しいか
- [ ] 本文が正しく表示されているか
- [ ] 見出し・段落の構造が崩れていないか
- [ ] サムネイル画像が表示されているか
- [ ] 関連記事が表示されているか
- [ ] CTAバナーが関連記事の手前に表示されているか

---

### STEP 5: デプロイ実行

**「デプロイ実行」** ボタンをクリック。  
完了まで **30秒〜2分程度** かかります。

完了後、画面に以下のメッセージが表示されます：
```
✅ デプロイ完了
公開URL: https://a-inc.info/insights/web-marketing/b-1/
```

## 7. デプロイ後の確認

### 確認する場所

| 確認先 | URL |
|--------|-----|
| デプロイした記事ページ | 完了画面に表示されたURL |
| 記事一覧ページ | https://a-inc.info/howto/posts/ |
| HOW TO BOX（新着欄） | https://a-inc.info/howto/ |

### ブラウザの強制リロード

`Cmd + Shift + R`（Mac）で強制リロードしてください。  
通常のリロードではキャッシュが残り、更新が確認できない場合があります。

### 確認ポイント

- [ ] 記事ページが正しく表示される
- [ ] 記事一覧に新しい記事が追加されている（最上位に表示）
- [ ] サムネイル画像・更新日・タグが正しく表示されている
- [ ] Notion のステータスが「掲載完了」に更新されている

## 9. よくあるエラーと対処法

### ❌ 「記事が一覧に表示されない」

**原因**: Notion のステータスが「デプロイ待ち」になっていない  
**対処**: Notion DB でステータスを確認・変更してから、ブラウザをリロード

---

### ❌ 「サムネイル画像が表示されない」（プレビュー）

**原因**: 画像が正しいフォルダに格納されていない、またはファイル名が違う  
**対処**: [第5章](#5-画像の準備と格納) の格納先・命名規則を再確認

---

### ❌ 「デプロイ後、一覧に反映されない」

**原因**: ブラウザのキャッシュ  
**対処**: `Cmd + Shift + R` で強制リロード。それでも反映されない場合は5分待ってから再確認

---

### ❌ 「エラー: TOC セクションが見つかりません」

**原因**: Notion 本文に `## TOC` セクションがない  
**対処**: Notion の本文フォーマットを確認し、`## TOC` セクションを追加

---

### ❌ 「ツールが起動しない / ブラウザで開けない」

**対処**:
1. ターミナルで `Ctrl + C` を押して停止
2. `npm start` で再起動
3. ブラウザで `http://localhost:3001` にアクセス

それでも解決しない場合は担当エンジニアに連絡してください。

---

## 付記：記事ステータスの流れ

```
未着手 → 画像待ち → 承認待ち → デプロイ待ち → 【デプロイ実行】 → 掲載完了
                                    ↑                                    ↓
                              （ツールに表示される）           （自動で更新される）
```

「差し戻し」は、デプロイ中にエラーが発生した場合に自動的に設定されます。  
その場合は Notion の「メモ1」欄にエラー内容が記録されています。
