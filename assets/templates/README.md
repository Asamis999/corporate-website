# 採用応募テンプレートファイル

このディレクトリには、採用応募用のテンプレートファイルを配置します。

## 必要なファイル

### 1. resume_template.pdf
履歴書のテンプレートファイル

**推奨内容：**
- 基本情報欄（氏名、生年月日、住所、連絡先）
- 学歴欄
- 職歴欄
- 免許・資格欄
- 志望動機欄
- 本人希望記入欄

### 2. career_template.pdf
職務経歴書のテンプレートファイル

**推奨内容：**
- 職務要約
- 職務経歴詳細（会社名、期間、職務内容）
- 活かせる経験・知識・技術
- 自己PR

## テンプレートファイルの作成方法

### オプション1: Wordで作成してPDF化
1. Microsoft WordやGoogle Docsで履歴書・職務経歴書のフォーマットを作成
2. PDF形式で保存
3. このディレクトリに配置

### オプション2: 既存のテンプレートを使用
1. 厚生労働省の履歴書様式などをダウンロード
2. 必要に応じてカスタマイズ
3. PDF形式で保存してこのディレクトリに配置

### オプション3: オンラインツールを使用
- [Canva](https://www.canva.com/) - 履歴書テンプレート
- [yagish](https://rirekisho.yagish.jp/) - 履歴書作成サービス

## ファイル配置後の確認

テンプレートファイルを配置したら、以下のURLでアクセスできることを確認してください：

- https://your-domain.com/assets/templates/resume_template.pdf
- https://your-domain.com/assets/templates/career_template.pdf

## 注意事項

- ファイル名は必ず `resume_template.pdf` と `career_template.pdf` にしてください
- ファイルサイズは5MB以下を推奨
- PDF形式で保存してください
