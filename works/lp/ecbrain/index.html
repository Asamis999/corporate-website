プロジェクト概要

BtoB集客戦略の設計からLP制作・LINE構築、広告・LPOによる継続改善まで一貫支援

中小製造業のEC展開を支援する「EC BRAIN」。本プロジェクトでは、ターゲット分析と集客戦略の設計 → WIXでのLP制作・LINE構築 → 広告運用・LPOによる継続改善まで一貫して支援。リード獲得率の改善を継続しています。

支援内容

Phase 1｜戦略設計

「ECを始めたいが何から手を付ければいいかわからない」「ECの売上が伸びない」という製造業特有の課題を整理。「EC運用の課題解決 × 成果に直結するサポート」を訴求軸に、LINE・広告を含む集客シナリオを策定しました。

Phase 2｜制作・構築

戦略に基づき、以下を実装しました。

・事例・実績データをファーストビューで強調し信頼感を醸成
・スマホ最適化とCTA複数配置でCVR向上
・LINE公式アカウントの構築とナーチャリング導線の設計
・WIXでクライアントが自身で更新できる運用設計

Phase 3｜継続的な運用・改善支援

リリース後も以下の運用支援を継続しています。

・LPOによるファーストビュー・CTAの最適化
・広告運用とリード獲得率の改善
・LINE配信シナリオの最適化
・SEO施策によるオーガニック流入の強化

デザインギャラリー

信頼性を高める実績データの可視化
課題解決型のサービス提案設計
データにインパクトを持たせるグラフィック生成

担当者コメント

製造業特有の課題を理解した上で、LP・LINE・広告・LPOを組み合わせたリード獲得基盤を構築しました。「作って終わり」ではなく、SEO施策・広告最適化・LP改善を継続し、リード獲得率の向上に伴走しています。
